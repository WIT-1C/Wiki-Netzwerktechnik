# Wiki: Netzwerktechnik — Index

Zielgruppe: IT-Auszubildende und Systemadministratoren

## Aufbau und Zweck

- Praktische Hinweise, Sicherheitsaspekte und Beispiele
- Klarer Pfad für weiterführende Detailseiten (siehe Inhaltsverzeichnis)
